Text indentation

‘def text_indentation(text)’ prints a text with 2 new lines after each of these characters: ., ? and :. Text is the content to be input.

Usage
=====
def text_indentation(text)’ prints a text with 2 new lines after each of these characters: ., ? and :. Text is the content to be input.

	>>>text_indentation == __import__(‘5-text_indentation’).text_indentation

Valid text
==========
Valid text should contain a string otherwise a TypError is raised.
::
	>>>text_indentation(“hi.”)
	hi.
	<BLANKLINE>

::
	>>>text_indentation(“good morning”)
	Good morning.
	<BLANKLINE>

There are no spaces at the beginning or end of the file.

::
	>>>text_indentation(“ good morning.”)
	Good morning.
	<BLANKLINE>

::

	>>>text_indentation(“good morning. ”)
	Good morning.
	<BLANKLINE>

::
	>>>text_indentation(“good morning. How are you? ”)
	Good morning.
	<BLANKLINE>
	How are you?
	<BLANKLINE>

New lines are only printed after ., ? and : characters
::
	>>>text_indentation(“good morning, the sun’s out”)
	good morning, the sun’s out

New lines between strings are treated as usual.
::
	>>>text_indentation(“good morning./nThe sun’s out.”)
	good morning.
	<BLANKLINE>
	<BLANKLINE>
	The sun’s out.
	<BLANKLINE>

Invalid Text
=============
If text is not a string, a TypeError is raised.
::
	>>>text_indentation(15)
	Traceback (most recent call last):
	TypeError: text must be a string
Text also has to be provided.
::
	>>>text_indentation()
	Traceback (most recent call last):
	TypeError: text_indentation missing 1 positional argument: text
